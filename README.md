## n8n-mcp-ai

Proyecto base para desarrollo con IA en Cursor.

### Estructura
- `apps/web`: Next.js + TypeScript (UI / API routes)
- `packages`: paquetes compartidos (placeholder)
- `python`: scripts/pipelines Python (placeholder)

### Requisitos
- Node.js LTS

### Inicio rápido
```bash
cd apps/web
npm install
cp .env.example .env.local
npm run dev
```

### Scripts útiles (apps/web)
- `npm run lint`
- `npm run typecheck`
- `npm run format`
